## 依赖关系

本任务列表中的许多任务相互依赖，因此我们需要按照适当的顺序执行它们：

1. 首先开发 WittCore 核心库和数据模型
2. 然后实现数据迁移工具
3. 接下来重构前端状态管理和 UI
4. 实现 Tauri 集成和 API
5. 开发 Anki 集成
6. 最后进行测试和发布

## 完成标准

所有任务完成后，系统应该能够：

1. 成功将旧数据迁移到新模型
2. 提供完整的 Note-Context 管理功能
3. 支持最多 5 个 Context 槽位
4. 实现与 Anki 的无缝集成
5. 提供强大的搜索和过滤功能
6. 支持媒体文件管理
7. 提供良好的用户体验
8. 具有可接受的性能
